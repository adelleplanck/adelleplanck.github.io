MY PROGRAMMING JOURNEY

As a little kid I was always into anything science, especially biology & chemistry, but when I started high school my interest almost completely shifted to the IT and Computer Science field. I learned Python for a while and switched to Java because I wanted to make desktop applications. I did a fundamentals course online and learned Swing & JavaFX afterwards.

On the side of doing Java, I stared learning basic HTML and CSS and played around with it a bit. Later I started incorporating some JavaScript into the websites. I can work with the Three.js library for 3D objects and I'm currently busy with a React course.
