HOW I GOT INTO PROGRAMMING

I was always into science and forensic biology. Then I quickly realised that it was way too hard for me and lost all interest in the basic sciences. I then discovered Computer Science and learned Python purely for the purpose of doing my homework. When I noticed how fun programming was I decided to learn Java. I did an online fundementals course and continued to experiment with different frameworks.


MY WEB DEVELOPMENT JOURNEY

After a few months of programming I decided that this was what I wanted to persue as a career. I did a bunch of research and came to the conclusion that it would benefit me to start learning how to make websites. With a lot of YouTube videos and help from StackOverflow, GeeksforGeeks & W3Schools, I started experimenting with different features, techniques and frameworks. Eventually I started compiling some of my favourite projects into this portfolio so that I can showcase my skills and creativity.
